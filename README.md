# Registro de Adquisiciones
Este proyecto es para registrar las adquisiciones, lo realizare con ExpressJS, MongoDB

Ejecutar en Ubuntu 
$ cd HISTORIA
$ npm install
$ DEBUG=myapp:* npm start

Ejecutar en MacOS OSX
$ cd HISTORIA
$ npm install
$ DEBUG=myapp:* npm start

Ejecutar en Windows
> cd HISTORIA
> npm install
> set DEBUG=myapp:* & npm start
